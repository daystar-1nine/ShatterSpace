# Shatter Space

A high-octane 2D space shooter with AAA glassmorphism UI, roguelite upgrades, and intense boss fights, powered by a custom WebGL hybrid rendering pipeline.

## Features
- **Hybrid Rendering:** Blends 2D Canvas gameplay with a rich 3D Three.js background, composited together for incredible performance and Bloom effects.
- **Roguelite Upgrades:** Earn credits, upgrade your ship, unlock new modules, and defeat increasingly difficult waves of alien dreadnoughts.
- **Cinematic Game Feel:** Heavy hit-stop freezes, screen shake, chromatic aberration, film grain, and camera lerp create an intensely satisfying arcade experience.
- **Full Gamepad Support:** Play entirely with an Xbox/PlayStation controller, including full UI navigation via a custom Virtual Cursor.
- **Optimized for Web:** Uses advanced Object Pooling to completely eliminate Garbage Collection micro-stutters during heavy bullet hell sequences.

## Technologies Used
- HTML5 Canvas (Gameplay logic)
- Three.js (Backgrounds, Particle Systems, Post-Processing)
- Howler.js (Audio Engine)
- Vite (Build Tool)

## How to Play
1. Install dependencies: \`npm install\`
2. Start the development server: \`npm run dev\`
3. Open http://localhost:5173 in your browser (Chrome/Edge recommended).
